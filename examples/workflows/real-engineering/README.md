# Real Engineering Workflow

This example combines:

- RTK command discipline
- Ponytrail file-change snapshots
- Superpowers brainstorming and writing-plans
- Matt Pocock engineering skills

Install Matt Pocock's skills before installing this workflow:

```bash
npx skills@latest add mattpocock/skills
```

Then run `/setup-matt-pocock-skills` in your agent app.

Validate this workflow from the Ponyrace repo:

```bash
bun run dev -- bundle validate examples/workflows/real-engineering
```

Install it into a project:

```bash
bun run dev -- workflow install examples/workflows/real-engineering
```
